`public` is a submodule pointing to https://github.com/jreisinger/jreisinger.github.io. See `.gitmodules`.

Usual workflow

* Modify `content` or `config.toml`
* [Check locally `hugo server`]
* Commit and push changes
* `./deploy.sh`
